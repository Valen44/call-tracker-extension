Call Tracker does not collect, store, transmit, or share any personal user data.

## Data Collection
The extension does not collect any personal information, financial data, authentication data, communications, browsing history, or website content.

## Local Processing
All data processing is performed locally within the user's browser.  
The extension only reads visible status text on supported portals to calculate call metrics and working time.

## Data Storage
Any data stored by the extension (such as call history or user settings) is saved locally in the browser using Chrome's local storage and is never transmitted to external servers.

## Third Parties
Call Tracker does not share data with third parties and does not use analytics, tracking, or advertising services.

## Changes
This privacy policy may be updated if the extension's functionality changes.

## Contact
If you have any questions about this privacy policy, you can open an issue in the GitHub repository.
